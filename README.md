# RR1Code
Code für einen 1-tägigen React-Redux-Kurs.

## Voraussetzungen:
* Node.js Version 6.x oder höher
* npm install -g create-react-app

## Erste Schritte:
* create-react-app my-app
* cd my-app
* npm start
* http://localhost:3000
